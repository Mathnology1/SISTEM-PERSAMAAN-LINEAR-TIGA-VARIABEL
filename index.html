📖 Materi: Sistem Persamaan Linear Tiga Variabel

🎯 Definisi

Sistem persamaan linear tiga variabel adalah sistem yang terdiri dari tiga persamaan linear dengan tiga variabel yang tidak diketahui. Bentuk umumnya:

ax + by + cz = d
ex + fy + gz = h
ix + jy + kz = l

🔧 Metode Penyelesaian

1. Metode Eliminasi

Menghilangkan satu variabel dengan menjumlahkan atau mengurangkan persamaan

2. Metode Substitusi

Mengganti variabel dengan nilai yang sudah diketahui

📋 Langkah-langkah Penyelesaian

  1. Buat model matematika dari soal cerita
  2. Susun sistem persamaan linear
  3. Eliminasi satu variabel untuk mendapat sistem 2 variabel
  4. Selesaikan sistem 2 variabel
  5. Substitusi kembali untuk mencari variabel ketiga
  6. Verifikasi jawaban

📝 Soal & Pembahasan: Toko Buah-buahan

📋 Soal

🍎 Di suatu toko buah-buahan:

  • • Ani membeli 2 apel dan 7 jeruk, membayar Rp42.000
  • • Asep membeli 4 apel dan 3 jeruk, membayar Rp40.000
  • • Budi membeli jeruk 4 buah lebih banyak dari apel, membayar Rp71.000

❓ Berapa banyak buah yang dibeli Budi?

Pilihan Jawaban:

A. 5 buah
B. 9 buah
C. 14 buah
D. 18 buah
E. 19 buah

🔍 Pembahasan Lengkap

Langkah 1: Membuat Model Matematika

Misalkan:

  • x = harga 1 apel (rupiah)
  • y = harga 1 jeruk (rupiah)
  • a = jumlah apel yang dibeli Budi
  • j = jumlah jeruk yang dibeli Budi

Dari informasi yang diberikan:

Ani: 2x + 7y = 42.000 ... (1)
Asep: 4x + 3y = 40.000 ... (2)
Budi: j = a + 4 ... (3)
Budi: ax + jy = 71.000 ... (4)

Langkah 2: Mencari Harga Apel dan Jeruk

Eliminasi variabel y dari persamaan (1) dan (2):

2x + 7y = 42.000 |×3| → 6x + 21y = 126.000
4x + 3y = 40.000 |×7| → 28x + 21y = 280.000
28x + 21y - (6x + 21y) = 280.000 - 126.000
22x = 154.000
x = 7.000

Substitusi x = 7.000 ke persamaan (1):

2(7.000) + 7y = 42.000
14.000 + 7y = 42.000
7y = 28.000
y = 4.000

Langkah 3: Mencari Jumlah Buah Budi

Dari persamaan (3): j = a + 4

Substitusi ke persamaan (4):

ax + jy = 71.000
a(7.000) + (a + 4)(4.000) = 71.000
7.000a + 4.000a + 16.000 = 71.000
11.000a = 55.000
a = 5

Maka jumlah jeruk Budi:

j = a + 4 = 5 + 4 = 9

Langkah 4: Jawaban Akhir

Total buah yang dibeli Budi:

5 apel + 9 jeruk = 14 buah

Jawaban: C. 14 buah

✅ Verifikasi

Ani: 2(7.000) + 7(4.000) = 14.000 + 28.000 = 42.000 ✓
Asep: 4(7.000) + 3(4.000) = 28.000 + 12.000 = 40.000 ✓
Budi: 5(7.000) + 9(4.000) = 35.000 + 36.000 = 71.000 ✓
